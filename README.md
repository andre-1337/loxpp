# Lox++

A modern, dynamically typed programming language built on the JVM (Java 11+).
Inspired by Bob Nystrom's "Lox", this implementation has been heavily extended to support advanced object-oriented paradigms, rich native data structures and non-blocking asynchronous programming.

## Features

Lox++ provides a massive host of improvements and additions over Lox:
* Pattern matching
* Default parameters
* Traits and abstract methods
* Static and private methods
* Namespaces and modules
* Rich data structures, such as arrays, tuples and dictionaries
* String manipulation with built-in native methods
* Asynchronous programming
* Big standard library, with support for structures such as `List`s, `Map`s, networking, runtime traits and others
* Sum types
* `try`/`catch`, with custom errors
* File imports
* `for`-`in` loops
